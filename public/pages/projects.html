Project Showcase: Full-Stack Community Management Portal for CEL Harrisonburg

www.celharrisonburg.org

Project Overview

I developed a comprehensive, full-stack web application from the ground up to serve as the digital backbone for CEL Harrisonburg, a professional organization for Latino entrepreneurs and business owners. The platform streamlines member management, automates event organization, and provides a secure, interactive hub for community engagement, replacing manual processes with an efficient, scalable digital solution.

As the sole developer, I was responsible for the entire project lifecycle, including system architecture, database design, backend development, frontend user-interface design, and security implementation.


Key Features & Technical Highlights

This application was built with a focus on security, real-time functionality, and ease of use for both members and administrators.

1. Secure Authentication & Role-Based Access Control

    I implemented a robust user registration and login system using Firebase Authentication, supporting both traditional email/password and Google OAuth for a seamless user experience. The system is built with role-based access, distinguishing between standard members and administrators, ensuring that sensitive management features are protected.


2. Custom-Built Admin Dashboard

    A key feature of the project is a comprehensive, secure admin panel. This dashboard empowers the CEL Harrisonburg team to:

  • Manage Business Approvals: Review new business submissions, with options to approve, deny, or edit their status.
  • Moderate Community Feedback: View and manage submissions from the "Comparte Tu Voz" form, categorized by status (Pending, In Progress, Complete).
  • Oversee User Data: View and manage the member directory and profile.

3. Real-Time Database & Content Management

    Using Cloud Firestore, the application's data is synchronized in real-time. This allows for dynamic features such as:

  • An interactive events calendar that is always up-to-date.
  • User profiles that can be edited and are instantly reflected across the site.
  • Live notification badges in the admin panel that update automatically when new items require review.

4. Secure File Management System

    I architected a secure file storage system using Firebase Storage for user profile pictures and business logos. This includes:

  • A structured, multi-level folder system that ties each file directly to both the userId and the specific businessId.
  • Strict Firebase Storage security rules that ensure a user can only upload or delete files within their own designated folders, preventing unauthorized access.

5. Third-Party API Integration via Serverless Backend

    To display a live Instagram feed without compromising security, I developed a serverless backend using Firebase Cloud Functions. The function acts as a secure intermediary, safely using a secret API token on the server-side to fetch posts and then passing the data to the frontend. This prevents exposing sensitive keys on the client-side.


Technology Stack

  • Frontend: Vue.js (Composition API), Vue Router
  • Backend & Database: Firebase (Cloud Firestore, Firebase Authentication, Firebase Storage, Cloud Functions)
  • Styling: Bootstrap 5, SCSS
  • Server-Side Logic: Node.js (for Cloud Functions)
  • Security: Firebase Security Rules, Google reCAPTCHA

Outcome

The result is a scalable, professional, and secure platform that empowers CEL Harrisonburg to manage its community efficiently and provides immense value to its members. The application serves as a central point for networking, resources, and communication, directly supporting the organization's mission to strengthen the local Latino business community.
